# App图标目录

这个目录用于存放PWA应用的各种尺寸图标。

## 📱 所需图标尺寸

PWA需要以下尺寸的图标才能在各种设备上正常显示：

- **72x72** - Android小图标
- **96x96** - Android中等图标
- **128x128** - Chrome Web Store
- **144x144** - Windows磁贴
- **152x152** - iOS标准尺寸
- **192x192** - Android标准尺寸
- **384x384** - Android大图标
- **512x512** - PWA启动画面

## 🎨 图标生成方法

### 方式一：使用生成脚本（推荐）

1. 准备一张512x512或更大的PNG图标
2. 运行生成脚本:

```bash
# 1. 安装Pillow库
pip install Pillow

# 2. 生成所有尺寸图标
python generate_icons.py your_logo_512x512.png
```

所有图标会自动生成到这个目录。

### 方式二：在线工具

1. 访问 https://www.pwabuilder.com/imageGenerator
2. 上传你的512x512图标
3. 下载生成的图标包
4. 解压到这个目录

### 方式三：手动制作

使用图片编辑软件（如Photoshop、GIMP）手动创建各个尺寸的图标。

## 🖼️ 图标设计建议

### 基本要求

- **格式**: PNG（支持透明背景）
- **形状**: 正方形
- **内容**: 简洁明了，易于识别
- **颜色**: 与应用主题一致

### 设计技巧

1. **简洁**: 图标应该简单，在小尺寸下也清晰可见
2. **对比**: 使用高对比度的颜色
3. **居中**: 主要元素居中显示
4. **边距**: 留适当边距，不要太满
5. **品牌**: 体现应用的品牌特色

### 推荐设计

对于物流视频录制系统，建议使用：

- 📦 包裹图标 + 📹 录像机图标组合
- 渐变紫色背景（#667eea → #764ba2）
- 白色前景图标
- 圆角矩形外框

## ✅ 检查清单

生成图标后，请检查：

- [ ] 所有8个尺寸的图标都已生成
- [ ] 文件命名正确（icon-{size}x{size}.png）
- [ ] 图标在浏览器中显示正常
- [ ] manifest.json中的路径正确
- [ ] iOS和Android设备上测试通过

## 🔗 相关文件

- `manifest.json` - 定义图标路径
- `index.html` - Apple Touch Icon引用
- `browserconfig.xml` - Windows磁贴配置

## 📝 注意事项

1. 图标文件应该小于100KB
2. 使用PNG格式，支持透明背景
3. 不要使用纯白色背景（iOS显示不佳）
4. 确保在深色和浅色背景下都清晰可见

---

**提示**: 如果暂时没有图标，系统会使用Emoji表情 📦 作为临时图标。建议尽快创建专业的图标以提升用户体验。
