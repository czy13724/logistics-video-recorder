# 📦 商家使用说明

## 🎯 给商家的话

这个软件是**专门为商家设计的物流视频录制工具**，无需任何技术知识，**下载后双击即可使用**。

---

## 💻 Windows用户（推荐）

### 下载安装
1. 从[发布页面](https://github.com/czy13724/logistics-video-recorder/releases)下载 **`物流视频管理-Windows.zip`**
2. 解压到任意文件夹（如桌面）
3. 打开文件夹，双击 **`启动.bat`** 或 **`物流视频管理.exe`**

### ⚠️ 首次启动提示
- **首次打开需要5-10秒**，这是正常的，请耐心等待
- 如果Windows提示"危险"或"不信任"，点击"更多信息" → "仍要运行"
- 之后每次启动会快很多

---

## 🍎 Mac用户

### 下载安装
1. 从[发布页面](https://github.com/czy13724/logistics-video-recorder/releases)下载 **`物流视频管理-macOS.dmg`**
2. 双击dmg文件打开
3. 将"物流视频管理.app"拖到"应用程序"文件夹
4. 在"启动台"或"应用程序"中找到并双击运行

### ⚠️ 首次启动提示
- **首次打开需要10-20秒**，会显示转圈，这是正常的，请耐心等待
- 如果提示"无法打开"：
  1. 打开"系统偏好设置" → "安全性与隐私"
  2. 点击"仍要打开"按钮
  3. 再次双击应用即可

---

## 🐧 Linux用户

### 下载安装
1. 从[发布页面](https://github.com/czy13724/logistics-video-recorder/releases)下载 **`物流视频管理-linux-x64.tar.gz`**
2. 解压：`tar -xzf 物流视频管理-linux-x64.tar.gz`
3. 添加执行权限：`chmod +x 物流视频管理`
4. 运行：`./物流视频管理`

---

## 📝 使用步骤

### 1. 启动软件
- Windows: 双击 `启动.bat`
- Mac: 双击应用图标
- Linux: 运行 `./物流视频管理`

### 2. 连接摄像头
- 插入USB摄像头
- 软件会自动检测

### 3. 开始录制
1. 扫描或手动输入快递单号
2. 点击"开始录制"
3. 拍摄包裹
4. 点击"停止录制"
5. 视频自动保存

### 4. 查看视频
- 点击"视频管理"查看所有录制
- 可以添加问题标记
- 可以导出报表

## ❓ 常见问题

### Q1: 软件打不开怎么办？

**Windows**:
- 右键exe文件 → 属性 → 解除锁定
- 或安装.NET Framework 4.8

**Mac**:
- 系统偏好设置 → 安全性与隐私 → 仍要打开
- 或终端运行: `sudo xattr -rd com.apple.quarantine /Applications/物流视频管理.app`

### Q2: 为什么启动很慢？

- **首次启动需要解压文件，需要5-20秒**
- 这是一次性的，之后会快很多
- 启动时请不要重复点击

### Q3: 摄像头无法使用？

- 检查USB连接
- 允许摄像头权限（系统会弹窗询问）
- 重启软件

### Q4: 视频保存在哪里？

- Windows: `软件目录/videos/`
- Mac: `应用程序/物流视频管理.app/Contents/videos/`
- 可以通过软件的"视频管理"查看

### Q5: 如何备份视频？

直接复制`videos`文件夹到U盘或云盘即可

### Q6: 需要联网吗？

**不需要！** 完全离线使用，所有数据存在本地
